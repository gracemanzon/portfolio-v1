02. featured projects

Dinosaur Game

A side scroller game utilizing the Phaser 2 framework and JavaScript. Inspired by the 1998 iMac game Nanosaur. Users can access basic movement and adhere to world boundaries and physics by using the arrow keys in browser.

InSnapx App

A portfolio app for retro camera lovers developed with full-stack Rails. Using the Nylas gem visitors can schedule consultations with artists and artists can connect their Google calendars to let their patrons know what upcoming events and conventions they will be attending.

eCommerce Sample

Stationary website mockup offering various products with relations to images, suppliers and purchasing carts. Backend includes user authentication and is built with Ruby on Rails. Frontend built with React featuring Bootstrap CSS.

Spotify Data Visualizer

Browser based app that connects a user to their Spotify data using the Spotify API. Users can save their snapshots in time which will allow reaccess to each timelime and features fun visual plots that compare their data. Backend built with Ruby on Rails and frontend built with React.
